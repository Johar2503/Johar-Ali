Console Functions

Console is used the view the log of the js development process and to display the page context.

There are three types of functions in console.They are:

1)console.log():This function is used display the message in the console and it is used for testing purpose.

2)console.error():This function is used display the error message in the console and it is used for testing purpose.

3)console.warn():This function is used display the warning message in the console and it is used for testing purpose.
