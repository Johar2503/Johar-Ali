Methods of String And Array

Remaining methods of String:

1)substring:It is used to get another string from the existing string.

2)splice:It is used to insert the items at the specified index in an array by removing or not removing the existing element at the given position.

3)replaceAll:It is used to replace the total string with the new string.

4)replace:It is used to replace the string with the desired letter or word.
